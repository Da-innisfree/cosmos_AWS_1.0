## 🚩개발목적

- React와 Spring Boot를 활용한 반응이 빠른 쇼핑몰
- 새로운 언어와 IDE를 사용하여 더 다양한 도전 시도
- 컴포넌트 단위로 세분화 시켜 개발

## 🛠 사용기술

`Frontend`
- HTML,CSS
- JavaScript
- React
- Axios
- Kakao Address API
- Kakao Pay API

`Backend`
- Java11
- Spring Boot
- Gradle
- MyBatis
- AWS EC2
- AWS RDS
- Nginx
- MariaDB
